打开终端并进入你想要初始化代码仓库的目录
运行以下命令初始化一个新的Git仓库：
git init
运行以下命令创建一个名为"question02.sh"的Shell脚本文件并打开编辑器：
touch question02.sh
vim question02.sh
在编辑器中输入以下命令并保存文件：
#!/bin/bash
git init
退出编辑器并运行以下命令将"question02.sh"文件添加到Git仓库中：
git add question02.sh
运行以下命令提交文件并添加提交消息：
git commit -m "Initialize new Git repository in current working directory"
