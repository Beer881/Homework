如果尚未创建GitHub帐户，请前往https://github.com并创建一个新帐户。

在GitHub中创建一个新的私有存储库。单击页面顶部的“+”按钮并选择“New repository”。然后，输入您的存储库名称，选择“Private”选项，并单击“Create repository”。

克隆您的存储库。在终端中，导航到您要存储库的目录中，然后运行以下命令：

git clone https://github.com/your-username/your-repo.git
创建，添加，提交和推送一个名为file1.txt的新文本文件。在终端中，导航到克隆的存储库的目录中，然后运行以下命令：


touch file1.txt
git add file1.txt
git commit -m "Add file1.txt"
git push origin main
在Gradescope中提交您的存储库。将存储库链接复制到Gradescope中的提交页面，然后单击“提交”。
